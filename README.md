### `[ MISSION 01 ]` — [Specter Proxy](https://github.com/abhi-byte62/specter-proxy)
> `Node.js` `HTTP/HTTPS` `TCP` `TLS` `Streams`

Local HTTP/HTTPS proxy for real-time traffic interception and network simulation. Intercepts, reads, and manipulates packets before they reach their destination — including TLS-encrypted sessions via a local CA.

```
  ✦  Packet interception via Node.js Streams with buffer management
  ✦  Latency injection, packet filtering, HTTP error simulation
  ✦  TLS inspection — MITMs its own encrypted traffic
  ✦  Live terminal dashboard for real-time analysis
```

### `[ MISSION 02 ]` — [Packet Sniffer 3D](https://github.com/abhi-byte62/packet-sniffer-3d-)
> `Three.js` `JavaScript` `Vite` `PCAP`

Real-time 3D network traffic visualizer. Makes packet flow across network topologies readable rather than a wall of logs — because if you can't see what's happening on the wire, you're guessing.

```
  ✦  PCAP parsing + localhost traffic monitoring with playback
  ✦  Hex payload viewer, TLS masking, configurable traffic controls
  ✦  Rendering pipeline optimized for high-frequency packet events
```
